# Android快速开发脚手架
整理了android开发中经常使用到的一些库

- 网络:
  - okhttp
  - retrofit
  - rxjava
  - com.squareup.okhttp3:logging-interceptor
  - rxjava适配器
  - json转换器
  - rxandroid
  - fastjson
- 代码简化:
	- lombok
	- butterknife
- 时间选择器:
	- pickerView
- 图片选择:
	- takephoto_library
- sql数据库ROM框架:
	- sugar
- fragment管理:
	- Fragmentation
- 下拉刷新:
	- SmartRefreshLayout
- 头部固定:
	- StickHeadScrollView
- web混合开发:
	- JsBridge
- 沉浸式状态栏:
	- ImmersionBar
- 消息传递
	- EventBus
